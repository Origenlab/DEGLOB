## 1. Proposito del documento y filosofia de marca

### 1.1 Objetivo central
Este documento establece el sistema oficial para crear, optimizar y mantener todas las secciones del sitio web de DEGLOB, garantizando:
- Contenidos 100% originales, sin copias ni parafraseos mecanicos.
- Redaccion profesional enfocada en ventas, emociones y conversion.
- Estructura web alineada con SEO avanzado y marketing estrategico.
- Diseno limpio, escalable y facil de mantener.

La aplicacion rigurosa de este manual asegura consistencia, calidad y posicionamiento competitivo en cada pagina.

### 1.2 Alcance
Aplica a:
- Nuevas paginas de servicios.
- Landing pages.
- Mejoras y reescrituras de paginas existentes.
- Paginas del blog y articulos (ver `DOCUMENTO-ARTICULOS.md`).
- Secciones visuales, estructurales, de codigo y contenido.

### 1.3 Filosofia de marca
DEGLOB representa profesionalismo, creatividad y servicio impecable. Cada pieza del sitio debe comunicar:
- Creatividad visual en montajes con globos.
- Direccion artistica profesional.
- Logistica precisa y resultados memorables.

**Pilares de comunicacion**

| Pilar | Aplicacion directa |
| --- | --- |
| Profesionalismo | Contenido claro, confiable y basado en experiencia real |
| Creatividad | Propuestas unicas, ideas inspiradoras y estilo diferenciado |
| Confiabilidad | Tono cercano, procesos claros y testimonios |
| Calidad premium | Enfoque en materiales, tecnica y resultados fotograficos |

### 1.4 Cliente ideal y objeciones
**Perfil ideal**
- Mujeres 25-45 anos.
- Organizadoras de eventos, planners, responsables de marketing.
- Buscan decoraciones memorables y listas para fotografiar.
- Valoran servicio profesional, puntualidad y estetica impecable.

**Objeciones comunes (resolver en contenido)**
1. Cuanto duran los globos.
2. Que incluye el servicio.
3. Cuanto tardan en montar.
4. Si se pueden personalizar colores.
5. Como reservar y con cuanta anticipacion.

## 2. Principios de originalidad y no plagio

### 2.1 Regla absoluta
Todo contenido se crea desde cero. Prohibido:
- Copiar textos de sitios externos.
- Parafrasear estructuras ajenas.
- Reutilizar textos identicos dentro del mismo sitio.

### 2.2 Como asegurar originalidad
1. Analizar referencias, pero escribir con lenguaje propio.
2. Adaptar cada texto a la intencion del usuario.
3. Evitar frases genericas o copiadas.
4. Cambiar enfoques, ejemplos y propuestas.

### 2.3 Uso de IA
- Siempre editar y personalizar.
- Solicitar contenido original desde el brief.
- Rechazar textos plantillados o sin valor.

### 2.4 Criterios de publicacion
Un texto puede publicarse solo si:
- Es original.
- Refuerza la marca.
- Cumple SEO.
- Tiene tono profesional y cercano.
- Incluye CTAs estrategicos.

## 3. Proceso general de desarrollo

### 3.1 Pasos principales
1. Definir proposito de la pagina.
2. Identificar palabras clave principales y secundarias.
3. Disenar la estructura de encabezados.
4. Redactar contenido original.
5. Optimizar SEO.
6. Ajustar diseno visual.
7. Integrar CTAs estrategicos.
8. Revisar experiencia movil.
9. Validar ortografia y estilo.
10. Publicar y medir desempeno.

### 3.2 Intensidad del contenido
- Pagina estandar: minimo 600 palabras.
- Servicios principales: 1000 a 1500 palabras.
- Landing pages premium: 1500 a 2500 palabras.

## 4. Arquitectura del documento HTML

### 4.1 Estructura minima obligatoria
```html
<!DOCTYPE html>
<html lang="es-MX">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <meta http-equiv="X-UA-Compatible" content="IE=edge" />
    <title>...</title>
    <meta name="description" content="..." />
  </head>
  <body>
    <!-- Contenido -->
  </body>
</html>
```

### 4.2 Reglas generales
- Usar estructura semantica correcta.
- Solo un H1 por pagina.
- Incluir favicon y meta tags completos.
- Usar imagenes WebP/AVIF cuando sea posible.
- Mantener `lang="es-MX"` en todo el sitio.

## 5. Optimizacion SEO avanzada

### 5.1 Palabras clave
Ejemplos base:
- decoracion con globos
- arcos organicos
- decoracion profesional para eventos
- montajes con globos en CDMX

Regla: definir una keyword principal y 3 a 6 secundarias por pagina.

### 5.2 Estructura de encabezados
- H1: tema principal + ubicacion si aplica.
- H2: secciones grandes.
- H3: detalles, listas o preguntas.

### 5.3 Meta title
- Maximo 60 caracteres.
- Incluir keyword principal al inicio.
- Terminar con la marca cuando aplique.

### 5.4 Meta description
- 140 a 160 caracteres.
- Incluir beneficio y enfoque local.
- CTA sutil al final.

### 5.5 Alt text
- Descriptivo y natural.
- Incluir keyword solo si es relevante.
- Evitar stuffing.

### 5.6 Densidad de keywords
- Entre 1.5% y 2.5%.
- Priorizar naturalidad sobre repeticion.

### 5.7 SEO on-page adicional
- Canonical correcto.
- OG y Twitter completos.
- URLs limpias y en kebab-case.
- Interlinking estrategico (servicios, eventos, blog).

## 6. Schema.org y datos estructurados

### 6.1 Tipos recomendados por pagina
- Organization
- LocalBusiness
- Service (servicios)
- BlogPosting o Article (blog)
- FAQPage (si hay FAQs)
- BreadcrumbList

### 6.2 Reglas
- JSON-LD en el `<head>`.
- Coherencia con el contenido real.
- Sin datos inventados (fechas, precios, reviews).

## 7. Sistema de diseno (CSS)

### 7.1 Principios visuales del sitio
- Elegante, moderno y confiable.
- Tipografia principal: Poppins.
- Maximo de ancho: 1400px.
- Uso de gradiente principal de marca.

### 7.2 Paleta base (alineada al sitio)
- Primario: #8B30EE
- Secundario: #C900D4
- Acento: #00AB1A
- Oscuros: #1a1a2e / #16213e
- Claros: #F8F9FA / #FFFFFF

### 7.3 Componentes clave
- Botones CTA con gradiente o WhatsApp.
- Cards con sombra y bordes suaves.
- Hero a dos columnas.
- Secciones con margenes amplios.
- Sidebar con interlinking (blog).

## 9. Jerarquia semantica

Ejemplo recomendado:
```
H1 - Decoracion con globos profesional en CDMX
  H2 - Servicios que ofrecemos
    H3 - Arcos organicos
    H3 - Backdrops
  H2 - Por que elegirnos
  H2 - Galeria
  H2 - Preguntas frecuentes
```

Reglas:
- No saltar niveles.
- H2 siempre para secciones mayores.
- H3 para subapartados.

## 11. Sistema de conversion

CTAs recomendados:
- Cotizar decoracion.
- Agendar asesoria.
- Ver proyectos.
- Cotizar por WhatsApp.

Reglas:
- CTA principal arriba del fold.
- CTA secundario dentro del contenido.
- CTA final antes del footer.
- Enlaces a WhatsApp personalizados por pagina.

## 12. Copywriting profesional

### 12.1 Tono
- Profesional.
- Cercano.
- Visual.
- Inspirador.

### 12.2 Reglas
- Usar segunda persona.
- Enfocar beneficios, no solo caracteristicas.
- Variar estructuras para evitar monotonia.
- Agregar emocion sin exagerar.

### 12.3 Formula base para parrafos clave
```
[Problema o deseo] + [Solucion DEGLOB] + [Beneficio claro] + [CTA sutil]
```

## 13. QA - Protocolo de calidad

Antes de publicar:
1. Revisar ortografia.
2. Revisar SEO (title, description, H1, keywords).
3. Revisar enlaces internos.
4. Revisar diseno movil.
5. Revisar originalidad.
6. Probar CTA de WhatsApp.
7. Verificar carga de imagenes.
